💡 How to load your test results:

  1. Run npx playwright test --reporter=html
  2. Open playwright-report/index.html in browser
  3. Press F12 → Console → type: copy(JSON.stringify(window.playwrightReport))
  4. Save as results.json and click "Load JSON File"
